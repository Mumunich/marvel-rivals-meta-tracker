# 🦸‍♂️ Marvel Rivals Meta Tracker

Простое и быстрое веб-приложение на **Streamlit**, которое в один клик достает актуальный тир-лист персонажей из игры Marvel Rivals. Больше не нужно блуждать по сайтам с рекламой — нажал кнопку и видишь мету.

## 🛠 Технологии
- **Python** — основа логики.
- **Streamlit** — современный веб-интерфейс без JS.
- **Playwright** — мощный движок для автоматизации браузера и скриншотов.

## 🚀 Как запустить у себя

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com
   cd marvel-rivals-tracker
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Установите браузер для Playwright:**
   ```bash
   playwright install chromium
   ```

4. **Запустите приложение:**
   ```bash
   streamlit run app.py
   ```

## 📂 Структура проекта
- app.py — основной интерфейс приложения.
- screenshoter.py — изолированный скрипт для парсинга (решает проблемы с asyncio на Windows).
- tierlist_final.png — временный файл скриншота (перезаписывается автоматически).

## 💡 Особенности
- Обход защиты сайтов через запуск в отдельном процессе.
- Красивый UI с использованием кастомного CSS.
- Ссылка на первоисточник данных (RivalsMeta).
